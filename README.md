# PLNet - Plackett Luce Networks
**PLNet** is a neural network based approach for preference learning.

The implementation is self contained and based on Matlab. Demo scripts and sample data are provided. Further information about the method can be obtained in the paper (see below).

## Reference
D. Schäfer and E. Hüllermeier, ‘Plackett-Luce networks for dyad ranking’, in Workshop LWDA, “Lernen, Wissen, Daten, Analysen”, Potsdam, Germany, (2016).
